# TypeScript Example — Placeholder

This directory is a placeholder for a future TypeScript example project generated using the Agent Swarm Programming Framework.

## Intended Project Type

A TypeScript/Node.js backend service or full-stack application demonstrating the framework's output for a web or API task — for example, a REST API with JWT auth, an AI integration service, or a real-time WebSocket server.

## How to Generate Your Own

1. Open Claude Code in any directory: `claude`
2. Paste the contents of `PROMPT_UNIVERSAL.md` from the repo root
3. Describe a TypeScript goal, for example:

   > "Create a TypeScript Express API with GET/POST /users endpoints, SQLite storage, input validation with Zod, and JWT authentication."

4. The swarm selects TypeScript, plans, architects, codes, tests, reviews, optimizes, and delivers production-ready code.

## What to Expect

The generated project will include:
- `src/` with strict TypeScript source code
- `tests/` with a Vitest or Jest test suite
- `package.json` and `tsconfig.json` with required configuration
- Usage instructions and known limitations
- A quality score (1–10) from the Best Practices Review Agent
